1.2.3 Berkeley Unix

GNU C Library определяет средства для некоторых версий Unix, которые формально не стандартизированы, в частности 4.2 BSD, 4.3 BSD, and 4.4 BSD Unix systems (также известные как Berkeley Unix) и SunOS (популярная производная от 4.2 BSD, которая включает некоторую функциональность Unix System V). Эти системы поддерживают большинство средств ISO C и POSIX, и 4.4 BSD и новые релизы SunOS по факту поддерживают их всех(in fact support them all)(?).

Средства BSD включают символьные ссылки (см. 14.5 Символьные ссылки), функцию select (см. 13.8 Ожидание В/В), сигнальые функции BSD (см. 24.10 Обработка сигналов BSD), и сокеты (см. 16 Сокеты).
