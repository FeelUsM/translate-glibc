1.1 Начало работы

Этот мануал написан в предположении, что вы покрайней мере немного знакомы с языком программирования C и основами программирования. В частности, подразумевается знакомство со стандартом ISO C (см. 1.2.1 ISO C), а не с “традиционными” до-ISO C диалектами.

The GNU C Library включает несколько заголовочных файлов, каждый из которых предоставляет объявления и определения для групп похожих средств; эта информация используется компилятором C когда обраатывается ваша программа. К примеру, заголовочный файл stdio.h определяет средства для ввода и вывода, а заголовочный файл string.h определяет утилиты обработки строк. Организация этого мануала в основном следует такому же разбиению как и заголовочные файлы.

Если вы читаете этот мануал в первый раз, вы должны прочесть весь вводный материал и просмотреть оставшиеся главы. Там достаточно функций в GNU C Library, и неоправданно ожидать, что вы в состоянии запомнить в точности, как использовать каждую из них. Более важно просто познакомиться с типами средств, которые предоставляет библиотека, так, что бы во время написания программ вы могди распознать, когда использовать функции библиотеки, и где в этом мануале вы можете найти более детальную информацию о них.
