1.2.2 POSIX (The Portable Operating System Interface)

The GNU C Library также совместима с набором стандартов ISO POSIX, известного более формально как Portable Operating System Interface for Computer Environments (ISO/IEC 9945). Они были дважды опубликованы как ANSI/IEEE Std 1003. POSIX - в основном производный от различных версий операционных систем Unix.

Средства библиотеки, специфицированные стандартами POSIX являются надмножеством средств, требуемых ISO C; POSIX определяет ополнительные возможности функций ISO C, а также определяет новые дополнительные функции. В общем, дополнительные требования и функциональность, определяемые в стандартах POSIX нацелены на предоставление низкоуровневой поддержки для различных типов операционных систем, а не для поддержки основного языка программирования, которая может работать на множестве разных сред операционных систем (?...).

The GNU C Library реализует все функции, заданные в ISO/IEC 9945-1:1996, the POSIX System Application Program Interface, также называемом POSIX.1. Основные расширения средств ISO C, задаваемые этим стандартом, включают примитивы интерфейса файловой системы (см. 14 Интерфейс файловой системы), Устройство-зависимые(device-specific) функции управления терминалом (см. 17 Низкоуровневый интерфейс терминала), и функции управления процессами (см. 26 Процессы).

Некоторые средства из ISO/IEC 9945-2:1993, the POSIX Shell and Utilities standard (POSIX.2) также реализованы в GNU C Library. Они включают утилиты для работы с регулярными выражениями и другие средства сопоставления с образцом (см. 10 Сопоставление с образцом).
